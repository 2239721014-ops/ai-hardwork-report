⌨️ 高级配置指令汇总

1. 版本检查与升级

# 查看 OpenClaw 版本
openclaw -v

# 升级 OpenClaw（需满足版本要求）
npm install -g openclaw

# 升级飞书官方插件到最新版本
npx -y @larksuite/openclaw-lark-tools update

⚠️ OpenClaw 版本要求：Linux/MacOS ≥ 2026.2.26，Windows ≥ 2026.3.2

2. 流式输出配置

# 开启流式输出
openclaw config set channels.feishu.streaming true

# 关闭流式输出

# 开启耗时展示
openclaw config set channels.feishu.footer.elapsed true

⚠️ 安全注意事项

  1. 核心风险：AI 能读取飞书数据，理论上存在泄露可能
  2. 强烈建议：个人账号先行试用，等安全隔离能力成熟后再用于工作环境
  3. 操作风险：AI 代发的飞书消息以用户名义发出，发出即成事实
  4. 最佳实践：重要操作"先预览，再确认"，避免"全自动驾驶"
