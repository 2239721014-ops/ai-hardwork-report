⚡ OpenClaw 命令大全

从入门到进阶 · 对话框常用命令
2026年3月更新
💡 快速上手： 复制下面的命令直接发送到对话框即可使用。大部分命令需要单独发送（不与其他内容混在一起）。
🆕 /new 新会话 📦 /compact 压缩 📊 /status 状态 🧠 /think 思考 🔄 /model 换模型 ⏹️ /stop 停止

🆕 新会话 & 重置 最常用

/new
开启一个全新会话
📌 场景：对话混乱了，想重新开始
/new minimax-m2.5
新建会话并指定模型
📌 场景：想用特定模型开始新对话
/reset
重置当前会话（同 /new）
📌 场景：功能同 /new，看你习惯
/compact
压缩上下文，保留关键信息
📌 场景：对话太长了，想让它更精简
/compact 保留xxx
压缩时指定保留的内容
📌 场景：想保留某个重要的上下文

📊 查看状态

/status
查看当前状态、用量、模型
📌 场景：想知道当前用的什么模型、还剩多少额度
/whoami
查看自己的用户 ID
📌 场景：添加白名单或调试时需要
/help
显示帮助信息
📌 场景：忘记命令了，看看快速参考
/commands
列出所有可用命令
📌 场景：想看看有哪些命令可以用

🧠 思考模式 进阶必学

/think high
开启深度思考模式
📌 场景：复杂问题需要更详细的推理
/think medium
中等思考深度（默认）
📌 场景：日常对话的标准模式
/think low
快速回答，减少思考
📌 场景：简单问题想要快速度回答
/think off
关闭思考模式
📌 场景：不需要思考，直接回答
/reasoning on
显示推理过程
📌 场景：想看 AI 是怎么思考的

🔄 切换模型

/model
查看当前使用的模型
📌 场景：不确定现在用的什么模型
/model list
列出所有可用模型
📌 场景：看看有哪些模型可以切换
/model minimax-m2.5
切换到指定模型
📌 场景：想用国产模型试试
/models
/model 的简写
📌 场景：和 /model 一样的作用

⏹️ 停止控制

/stop
停止当前正在生成的回复
📌 场景：AI 开始跑偏了，想让它停下
/restart
重启 Gateway 服务
📌 场景：遇到异常，重启服务试试

🔊 输出设置

/verbose on
开启详细输出（显示工具日志）
📌 场景：调试或想看 AI 做了什么
/verbose off
关闭详细输出（默认）
📌 场景：恢复正常简洁输出
/usage cost
显示本次对话花费
📌 场景：想看看花了多少钱

🤖 Agent 控制 进阶

/focus @agent
绑定当前对话到指定 Agent
📌 场景：在群聊中指定某个 Agent 响应
/unfocus
解除 Agent 绑定
📌 场景：不再需要指定 Agent 响应
/agents
查看有哪些可用 Agent
📌 场景：想知道能切换到哪些 Agent
/subagents list
查看当前子 Agent 状态
📌 场景：后台有任务在跑，看看进度
/kill all
终止所有子 Agent
📌 场景：想取消所有后台任务

💬 通道切换 进阶

/dock-telegram
切换回复到 Telegram
📌 场景：想从其他渠道切换到 Telegram 收消息
/dock-discord
切换回复到 Discord
📌 场景：想从其他渠道切换到 Discord 收消息

🔐 权限控制 进阶

/elevated on
开启提权模式（跳过确认）
📌 场景：需要执行敏感操作，减少确认弹窗
/elevated ask
询问模式（默认）
📌 场景：每次操作前确认
/elevated off
关闭提权
📌 场景：禁止执行危险操作

⚙️ 配置管理 高级

/config show
显示当前配置
📌 场景：查看完整配置
/config get xxx
查看指定配置项
📌 场景：查看某个具体配置
/allowlist add @xxx
添加到白名单
📌 场景：授权新用户使用命令
📌 常见问题：
• 命令怎么用？ 直接在对话框输入 /命令 然后发送
• 命令没反应？ 确保单独发送，不要和其他文字混在一起
• 模型切换在哪？ 用 /model minimax-m2.5 切换到 MiniMax
