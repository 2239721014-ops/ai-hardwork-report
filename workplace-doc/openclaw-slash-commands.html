💬 OpenClaw 对话框命令

Slash Commands
2026年3月更新
核心 会话 Agent 模型 输出 通道 配置
💡 提示：命令需要作为**独立消息**发送（以 / 开头），部分命令需要发送者被授权
⭐ 常用命令： /new 新会话 · /compact 压缩上下文 · /status 查看状态 · /help 帮助

⭐ 核心命令

/new [model]新建会话，可选指定模型
/reset重置会话（同 /new）
/compact [instructions]压缩上下文，保留要点
/stop停止当前运行
/restart重启 Gateway
/help显示帮助信息
/commands列出所有命令
/status查看当前状态
/whoami查看自己的 ID
/id/whoami 的别名

🔄 会话控制

/think <level>思考深度：off/minimal/low/medium/high/xhigh
/thinking/think 的别名
/t/think 的短别名
/verbose <mode>详细输出：on/full/off
/v/verbose 的别名
/reasoning <mode>推理显示：on/off/stream
/reason/reasoning 的别名
/elevated <mode>提权模式：on/off/ask/full
/elev/elevated 的别名
/usage <mode>用量显示：off/tokens/full/cost
/queue <mode>队列模式控制
/session idle <duration>设置空闲超时
/session max-age <duration>设置最大会话时长
/send on|off|inherit控制消息发送
/activation mention|always群聊激活模式

🤖 Agent 控制

/acp spawn启动 ACP 会话
/acp cancel取消 ACP 运行
/acp steer引导 ACP 方向
/acp close关闭 ACP 会话
/acp status查看 ACP 状态
/acp sessions列出 ACP 会话
/acp set-mode设置模式：persistent/oneshot
/subagents list列出子 Agent
/subagents kill终止子 Agent
/subagents log查看子 Agent 日志
/subagents info子 Agent 详情
/subagents send发送消息给子 Agent
/subagents steer引导子 Agent
/subagents spawn创建子 Agent
/agents列出线程绑定的 Agent
/kill <id|#|all>立即终止 Agent
/steer <id> <msg>引导运行中的 Agent
/tell/steer 的别名
/focus <target>绑定线程到 Agent
/unfocus解除线程绑定

🧠 模型选择

/model显示当前模型
/model list列出可用模型
/model <name>切换到指定模型
/models/model 的别名
/<alias>使用模型别名快速切换

🔊 输出控制

/tts off关闭语音
/tts always始终语音播报
/tts inbound仅播报输入
/tts tagged仅播报标记消息
/tts status查看 TTS 状态
/tts provider查看 TTS 提供商
/tts limit设置播报限制
/voiceDiscord 语音控制

💬 通道切换

/dock-telegram切换到 Telegram 回复
/dock_discord/dock-telegram 别名
/dock-discord切换到 Discord 回复
/dock_discord/dock-discord 别名
/dock-slack切换到 Slack 回复
/dock_slack/dock-slack 别名

⚙️ 配置管理

/config show显示配置
/config get获取配置项
/config set设置配置项
/config unset删除配置项
/debug show显示调试配置
/debug set设置调试项
/debug unset删除调试项
/debug reset重置调试配置
/allowlist add添加到白名单
/allowlist remove从白名单移除
/allowlist list列出白名单
/approve处理执行审批
/context查看上下文详情
/export-session导出会话
/export/export-session 别名

⚡ 执行命令

/exec查看当前执行设置
/exec host=<mode>设置执行主机
/exec security=<mode>设置安全模式
/exec ask=<mode>设置询问模式
/bash <command>执行 shell 命令
! <command>/bash 的快捷方式
!poll检查命令输出
!stop停止运行中的命令

🛠️ 技能命令

/skill <name>运行指定技能
/skill <name> <input>带输入运行技能
/proseOpenProse 写作助手
💡 使用技巧：
• 命令和参数之间可用 : 分隔，如 /think: high
• /new 后面可指定模型别名，如 /new minimax-m2.5
• 授权用户发送的纯命令消息会快速执行（跳过队列）
