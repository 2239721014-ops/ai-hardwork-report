安装步骤

1

克隆项目

git clone https://github.com/BeliefanX/ClawDeck.git
cd ClawDeck
2

安装依赖

npm install --legacy-peer-deps
3

配置环境

cp .env.example .env
💡 提示： 默认配置已连接到本地 OpenClaw，无需修改。
4

启动服务

# 终端1 - 前端 (端口6116)
npm run dev:client

# 终端2 - 后端 (端口6117)
npx tsx server/index.ts

✅ 访问 http://localhost:6116/

如何卸载？

删除文件夹即可，不影响 OpenClaw：

# 停止服务 (Ctrl+C)
# 删除文件夹
rm -rf ~/ClawDeck
