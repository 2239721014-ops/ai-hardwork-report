🎯 什么是 Command Palette？

Command Palette（命令面板）是 OpenClaw Control UI v2026.3.12 新增的快捷操作入口。它整合了所有常用命令和导航功能，让你无需切换到终端就能完成各种操作。

🔍

Command Palette 示意图
按 ⌘K 打开搜索面板

🚀 快速开始

1

打开 Control UI

在浏览器中访问 http://127.0.0.1:18789/ 或使用 openclaw dashboard 命令自动打开。

2

激活 Command Palette

按下 ⌘K (Mac) 或 CtrlK (Windows/Linux)，或者点击顶部的"搜索"按钮。

3

输入命令或搜索

直接输入命令（如 /new、/model）或搜索功能名称，然后按 Enter 执行。

📋 可用命令列表

💬 会话控制

/new 开始新会话
/reset 重置当前会话
/compact 压缩会话上下文
/stop 停止当前运行
/clear 清空聊天记录

⚡ 模式切换

/focus 切换专注模式
/think 设置思考级别
/verbose 切换详细输出
/fast 切换快速模式

ℹ️ 信息查看

/model 查看或设置模型
/status 查看会话状态
/usage 查看 Token 用量
/export 导出会话为 Markdown
/help 显示可用命令

🤖 代理管理

/agents 列出可用代理
/kill 终止子代理
/skill 运行技能
/steer 引导子代理

⌨️ 键盘快捷键

⌘ + K 打开 Command Palette
↑ ↓ 导航选项
Enter 执行选中命令
