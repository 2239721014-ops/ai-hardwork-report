🤖 openclaw acp 新增

ACP (Agent Control Protocol) 桥接工具，用于运行外部编码 Agent，如 Codex、Claude Code、Pi 等。

基本用法

# 交互式 ACP 客户端 openclaw acp client # 指定会话标签 openclaw acp --session-label my-agent acp client # 详细日志模式 openclaw acp -v client

常用选项

--session <key> 指定会话 key
--session-label <label> 指定会话标签
--provenance <mode> 来源模式：off / meta / meta+receipt
--reset-session 使用前重置会话
--require-existing 会话不存在时失败

👥 openclaw agents

管理隔离的 Agent，包括工作区、认证信息和路由绑定。

常用命令

# 列出所有 Agent openclaw agents list # 添加新 Agent openclaw agents add <agent-id> # 删除 Agent openclaw agents delete <agent-id> # 列出路由绑定 openclaw agents bindings # 添加路由绑定 openclaw agents bind <agent-id> --pattern "help/*" # 移除路由绑定 openclaw agents unbind <agent-id> --pattern "help/*" # 更新 Agent 身份信息 openclaw agents set-identity <agent-id> --name "我的助手" --emoji 🤖

🧠 openclaw models

模型发现、扫描和配置管理。支持 OpenRouter、OpenAI、Anthropic 等多种 provider。

常用命令

# 查看当前模型配置 openclaw models status # 列出可用模型 openclaw models list # 扫描 OpenRouter 免费模型 openclaw models scan # 设置默认模型 openclaw models set minimax-m2.5 # 设置图像模型 openclaw models set-image <model-alias>

内置模型别名

glm → zai/glm-5
minimax-m2.5 → MiniMax-M2.5
minimax-m2.5-highspeed → MiniMax-M2.5-highspeed
minimax-m2.5-lightning → MiniMax-M2.5-Lightning

管理别名和回退

# 管理模型别名 openclaw models aliases list openclaw models aliases add my-model "provider/model-name" openclaw models aliases remove my-model # 管理回退模型 openclaw models fallbacks list openclaw models fallbacks add <model-alias> openclaw models fallbacks remove <model-alias>

🐳 openclaw sandbox

Docker 沙箱容器管理，用于 Agent 隔离运行，确保安全性和可重复性。

常用命令

# 列出所有沙箱容器 openclaw sandbox list # 仅列出浏览器容器 openclaw sandbox list --browser # 重建所有容器 openclaw sandbox recreate --all # 重建特定会话容器 openclaw sandbox recreate --session main # 重建 Agent 容器 openclaw sandbox recreate --agent mybot # 解释沙箱/工具策略 openclaw sandbox explain

🔐 openclaw security

本地安全审计工具，检查 SSH、防火墙、更新状态等常见配置问题。

常用命令

# 运行安全审计 openclaw security audit # 深度检查（含 Gateway 探测） openclaw security audit --deep # 自动修复安全问题 openclaw security audit --fix # JSON 格式输出 openclaw security audit --json
🛡️ 审计项目： SSH 权限 · 防火墙状态 · 系统更新 · OpenClaw 配置安全 · 敏感文件权限

⬆️ openclaw update

更新管理和频道切换，支持 stable、beta、dev 三个更新频道。

常用命令

# 检查并更新 openclaw update # 查看更新状态 openclaw update status # 切换更新频道 openclaw update --channel stable openclaw update --channel beta openclaw update --channel dev # 预览更新（不执行） openclaw update --dry-run # 指定版本更新 openclaw update --tag beta # 更新后不重启 openclaw update --no-restart # 非交互模式 openclaw update --yes # 交互式更新向导 openclaw update wizard

🌐 Gateway 管理

openclaw gateway status 查看 Gateway 状态
openclaw gateway start 启动 Gateway
openclaw gateway stop 停止 Gateway
openclaw gateway restart 重启 Gateway
openclaw gateway --force 强制重启（先杀占用进程）
