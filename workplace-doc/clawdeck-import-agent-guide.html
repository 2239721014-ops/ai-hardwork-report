导入/配置智能体

1

打开智能体页面

在左侧导航栏点击 "智能体" (Agents) 进入管理页面。

2

理解智能体来源

ClawDeck 通过 OpenClaw Gateway 连接智能体。配置在 OpenClaw 端进行：

配置位置说明
~/.openclaw/OpenClaw 工作目录
AGENTS.md智能体定义文件
3

通过 AGENTS.md 导入智能体

在 OpenClaw 工作目录中创建或编辑 AGENTS.md：

# AGENTS.md 示例
---
name: my-agent
model: claude-sonnet
description: 我的私人AI助手
---
💡 提示： 修改后需重启 OpenClaw Gateway。
4

重启 Gateway

openclaw gateway restart
5

在 ClawDeck 中使用

  • 💬 聊天页面 - 选择智能体开始对话
  • 🔄 会话页面 - 管理智能体会话
  • ⏰ 定时任务 - 为任务指定智能体
❓

常见问题

Q1: 为什么看不到定义的智能体？

排查：
  • 确认 AGENTS.md 格式正确
  • 重启 OpenClaw Gateway
  • 刷新 ClawDeck 页面

Q2: 如何切换不同智能体？

在聊天页面或会话页面选择即可切换。

Q3: ClawDeck 和 OpenClaw 关系？

ClawDeck 是 OpenClaw 的 Web UI 管理界面
