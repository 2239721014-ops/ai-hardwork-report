📝 技能简介

Self-Improving + Proactive Agent 是一个具有自我反思、自我批评、自我学习和自我组织记忆能力的 AI 助手。它能够评估自己的工作、发现错误并进行永久性改进。

🧠 学习机制
学习触发条件
触发类型 示例 置信度 行动
直接纠正 "No, do X instead" 高 立即记录
重复提醒 "I told you before..." 高 标记重复，提高优先级
明确偏好 "Always/Never do X" 确认 升级为偏好
3次重复 同一纠正出现3次 确认 请求用户确认
项目限定 "For this project..." 作用域 写入项目级
自我反思 完成工作后评估 中等 记录lesson
🔒 安全边界
❌ 永不存储
  • 凭证（密码、API keys）
  • 金融信息（银行卡、加密货币）
  • 医疗数据
  • 生物特征
  • 第三方信息
  • 位置模式（家庭/工作地址）
✅ 透明度要求
  • 用户可随时查询"你知道我什么？"
  • 每条记录标注来源和时间
  • 删除后确认
  • 无隐藏状态
