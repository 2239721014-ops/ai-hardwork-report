  • 💰超低成本 - 按需运行容器，月费用约 $1（Free Tier 内仅 $0.23）
  • ⚡预热机制 - EventBridge 定时预热容器，零冷启动
  • 🌐Web 聊天 UI - React SPA 实时聊天界面
  • 💬Telegram 机器人 - 通过 Telegram 与 AI 助手对话
  • 🤖多 LLM 支持 - Claude、GPT、DeepSeek 等
  • 🔒安全认证 - AWS Cognito JWT 认证
  • 📦一键部署 - 单个 cdk deploy 命令完成

🏗️ 架构概览

前端 S3+CloudFront → API Gateway → Lambda → ECS Fargate → OpenClaw 容器

✅ 优点

  • 成本极低
  • 无需管理服务器
  • Web UI + Telegram 双接口
  • 自动扩缩容

  • 配置复杂，需要懂 AWS
  • Alpha 阶段
  • 国内访问可能慢
